# Project Summary

## Project Name
PrithviSaarthi - Climate Data Analytics Platform

## Overview
PrithviSaarthi is a climate analytics platform designed to analyze global environmental trends using Power BI dashboards and interactive visualizations.

The project focuses on understanding climate change through data-driven insights generated from environmental indicators such as greenhouse gas emissions, carbon dioxide concentration, renewable energy adoption, global temperature rise, and sea level variations.

## Objectives
- Analyze climate indicators over time
- Study greenhouse gas emission patterns
- Track global temperature changes
- Monitor renewable energy adoption
- Generate actionable climate insights

## Technologies Used
- Power BI
- Microsoft Excel
- Data Visualization
- Business Intelligence

## Key Insights
- Continuous increase in atmospheric CO₂ concentration
- Rising global temperature trends
- Growth in renewable energy adoption
- Sector-wise greenhouse gas emission patterns
- Long-term sea level rise observations

## Outcome
Developed an interactive climate analytics dashboard that simplifies environmental data and supports climate awareness through visual storytelling.
